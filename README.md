<H1> Classificação de Espécies de Flores com o Dataset Iris </H1>

Este projeto apresenta o desenvolvimento de um modelo de classificação supervisionada utilizando o clássico dataset Iris, amplamente empregado no ensino e na validação de algoritmos de Machine Learning.

O objetivo é demonstrar, de forma prática, o pipeline completo de um problema de classificação, desde a exploração dos dados até o treinamento e avaliação do modelo.

<H2> Aprendizados </H2>

* Implementação de modelos de classificação
* Uso prático do Scikit-learn
* Avaliação de modelos preditivos
* Interpretação de métricas de desempenho
* Estruturação de projetos de Machine Learning em notebooks
